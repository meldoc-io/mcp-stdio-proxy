System and account requirements for Meldoc MCP.

## System Requirements

- **Node.js:** 18.0.0 or newer
- **npm:** Included with Node.js
- **Claude Desktop:** Latest version recommended
- **Operating System:** macOS, Windows, or Linux

> ðŸ’¡ **Checking Node.js:** Open a terminal and run `node --version`. If the command is not found, install Node.js from <https://nodejs.org>

## Account Requirements

- **Meldoc Account:** Create at <https://app.meldoc.io>
- **Workspace Access:** At least one workspace
- **Permissions:** Read access minimum, write access for document operations

## Network Requirements

- **Internet Connection:** Required for API access
- **API Endpoint:** `https://api.meldoc.io` must be accessible
- **Firewall:** Must allow outbound HTTPS connections
